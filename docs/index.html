v0.dev AI Chat & Development Platform

Next-gen AI chat application and workspace powered by Together AI & Groq • Built with Next.js, TypeScript, and Tailwind CSS

View on GitHub Live Demo

✨ Key Features

  • 💬 Real-Time AI Concierge: Streaming chat sessions with Together AI & Groq models.
  • 🔄 Model Switching: Instantly toggle between Llama-3, Mixtral, DeepSeek Coder & more.
  • 📜 Version History & Diffing: Track, compare, and restore every generation.
  • 🛠️ Advanced Code Editor: Monaco editor with syntax highlighting & inline AI suggestions.
  • 📁 Project Management: Multiple workspaces, file trees, and deployment targets.
  • 🚀 One-Click Deployment: Push to Vercel or any Next.js host.
  • 🌈 Responsive UI: Tailwind CSS + shadcn/ui components for mobile & desktop.
  • 🔧 TypeScript & Tailwind: Type-safe code and utility-first styling.
  • ⚙️ Architecture Insights: AI-driven system design recommendations.

🚀 Quick Start

  1. Clone the repo
    git clone https://github.com/likhonsdev/v0.dev.git
cd v0.dev
  2. Install dependencies
    npm install
# or
yarn install
  3. Configure environment
    Create .env.local: 
    TOGETHER_API_KEY=your-together-key
GROQ_API_KEY=your-groq-key
NEXTAUTH_SECRET=supersecret
DATABASE_URL=postgresql://user:pass@host:port/db
  4. Run the server
    npm run dev
# or
yarn dev
    Then open localhost:3000.
